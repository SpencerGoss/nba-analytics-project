<objective>
Wire pace-normalized efficiency metrics into the team game feature pipeline by extending the opponent self-join, computing ten per-game advanced metrics from box score data, and rolling them with the existing shift-1 helper.

Purpose: ORtg, DRtg, eFG%, TS%, pace, and turnover rate per possession are pace-normalized metrics that remove confounding from raw box score counts. Rolling these per-game values gives the model pre-game efficiency signals that are comparable across eras and game paces.

Output: Updated `team_game_features.py` with per-game advanced metrics and rolling columns appearing in `team_game_features.csv`.
</objective>
