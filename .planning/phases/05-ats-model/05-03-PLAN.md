**Step A: Regenerate calibrated model artifact.**

Run `python src/models/calibration.py` to regenerate the calibrated artifact. If calibration.py fails due to the sklearn `cv='prefit'` issue (already resolved in Phase 1 via _CalibratedWrapper), verify the fix is in place before running.

**Step B: Create `src/models/value_bet_detector.py`:**

1. **Config:**
   - `VALUE_BET_THRESHOLD = float(os.getenv("VALUE_BET_THRESHOLD", "0.05"))` (5pp default, same as WINPROB_FLAG_PP)
   - Import from project modules: `_load_game_outcome_model` from game_outcome_model.py, `american_odds_to_implied_prob` from fetch_odds.py

2. **`no_vig_prob(home_ml, away_ml) -> tuple[float, float]`:**
   - Convert both sides to raw implied probability using american_odds_to_implied_prob()
   - Divide each by total (removes vig via multiplicative method)
   - Return (home_no_vig, away_no_vig)
   - Handle NaN/None moneylines: return (NaN, NaN) if either side is missing

3. **`check_remaining_quota(min_remaining=50) -> int`:**
   - Make a single cheap API call to `/sports` endpoint (1 credit) to read remaining quota
   - Parse `x-requests-remaining` header
   - If below `min_remaining`, raise a QuotaError (custom exception class in this file)
   - Return remaining credits
   - Handle missing API key gracefully: if ODDS_API_KEY is empty, skip quota check and return -1 with a warning

4. **`detect_value_bets(games_df, threshold=VALUE_BET_THRESHOLD) -> pd.DataFrame`:**
   - Input: DataFrame with columns including model_win_prob (from calibrated model) and market_implied_prob (from no-vig odds conversion)
   - Compute `edge = model_win_prob - market_implied_prob`
   - Flag rows where `abs(edge) > threshold`
   - Add columns: `edge`, `is_value_bet`, `bet_side` ("home" if edge > 0 else "away"), `edge_magnitude`
   - Return full DataFrame with value-bet annotations
   - Print summary: N value bets found out of M games, average edge magnitude

5. **`run_value_bet_scan(use_live_odds=True) -> list[dict]`:**
   - High-level function for daily use:
     a. If `use_live_odds=True`: call `check_remaining_quota()` first, then `fetch_game_lines()` from fetch_odds.py for current-season upcoming lines
     b. If `use_live_odds=False`: load from `game_ats_features.csv` (historical mode for testing)
     c. Load calibrated win-probability model
     d. For each game: compute model win probability, compute no-vig market implied probability
     e. Call `detect_value_bets()` on the combined DataFrame
     f. Return list of value-bet dicts (JSON-serializable for NFR-3)
   - Handle missing ODDS_API_KEY: print warning and fall back to historical mode

6. Make runnable: `if __name__ == "__main__"` runs `run_value_bet_scan(use_live_odds=False)` (historical mode -- safe to run without API key).

7. ASCII only in print statements.
