```python
CRAWL_DELAY = 3  # seconds — NFR-2 + robots.txt mandate
RETRY_DELAY = 6  # 2x crawl delay for failed requests (per RESEARCH.md anti-pattern avoidance)
MAX_RETRIES = 3
HEADERS = {"User-Agent": "Mozilla/5.0 (Windows NT 10.0; Win64; x64; rv:120.0) Gecko/20100101 Firefox/120.0"}

def get_referee_crew_assignments(
    start_date: str,
    end_date: str,
    output_dir: str = "data/raw/external/referee_crew/",
) -> pd.DataFrame:
    """
    Fetch referee crew assignments for NBA games in [start_date, end_date].

    Steps:
    1. Get list of game IDs for the date range from Basketball Reference schedule/scores pages
    2. For each game, fetch the box score page and extract officials
    3. Save results to CSV in output_dir
    4. Return combined DataFrame

    Rate limiting: sleeps CRAWL_DELAY seconds BEFORE every request (not after).
    Retry logic: on HTTP error or parse failure, sleep RETRY_DELAY and retry up to MAX_RETRIES.

    Args:
        start_date: "YYYY-MM-DD" format
        end_date: "YYYY-MM-DD" format
        output_dir: where to save CSV output

    Returns:
        DataFrame with columns: game_date, game_id_bref, home_team, away_team,
                                referee_1, referee_2, referee_3
    """
```

Implementation details:
- Use the selector confirmed in Task 1's spike test
- Basketball Reference game IDs follow format like "202501010LAL" — include this as `game_id_bref` column
- Also include `game_date`, `home_team`, `away_team` for downstream joining
- To discover game URLs for a date range, use the Basketball Reference scores page:
  `https://www.basketball-reference.com/boxscores/?month={M}&day={D}&year={Y}`
  which lists all games for a given date with links to individual box scores
- Extract referee names as separate columns: referee_1, referee_2, referee_3 (NBA uses 3 officials per game)
- Handle the case where some box scores might not have officials data (early seasons, etc.) — return NaN for missing refs
- `time.sleep(CRAWL_DELAY)` BEFORE every HTTP request — never after, never skip
- On HTTP 429 (rate limit): sleep 60 seconds and retry (per RESEARCH.md Pitfall 2)
- Save output as `referee_crew_{start_date}_{end_date}.csv` in output_dir
- Print progress: "Fetching game N of M: {game_id}..."
- Print summary at end: "Fetched {N} games, {M} with referee data, {K} missing"

Add a `__main__` block that demonstrates usage:
```python
if __name__ == "__main__":
    import sys
    start = sys.argv[1] if len(sys.argv) > 1 else "2025-01-01"
```

IMPORTANT anti-patterns from RESEARCH.md:
- Do NOT add this scraper to update.py — it's on-demand only (violates NFR-2 if in daily pipeline)
- Do NOT parallelize requests — one at a time with CRAWL_DELAY
- Do NOT save with `html.parser` — use `lxml` (faster, handles malformed HTML)
- Always sleep BEFORE the request, not after
