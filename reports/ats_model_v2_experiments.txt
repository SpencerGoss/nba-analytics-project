ATS MODEL V2 EXPERIMENTS REPORT
=================================================================
Generated: 2026-03-03
Branch: v2/ats-model-v2

OBJECTIVE
-----------------------------------------------------------------
Improve ATS holdout accuracy from 51.20% (v1 baseline) toward
52.4% breakeven threshold at -110 standard vig.

APPROACH
-----------------------------------------------------------------
1. Feature Engineering: Added 12 new ATS-specific features
2. Model Alternatives: Tested 6 model types (LR, L1-LR, GB, GB-conservative, RF, MLP)
3. Feature Selection: Permutation importance + bottom-30% pruning
4. Ensemble: Averaged LR + GB probabilities with threshold tuning
5. Confidence Filtering: Tested selective betting by model confidence

KEY FINDINGS
-----------------------------------------------------------------
1. The v2 model improved holdout accuracy from 51.20% to 51.41%,
   a gain of +0.21 percentage points. This is a marginal improvement
   that does NOT cross the 52.38% breakeven threshold.

2. Logistic regression remains the best model type. Tree-based models
   (GB, RF) and neural nets (MLP) consistently underperformed in
   expanding-window validation. ATS prediction lacks the strong
   signal structure that tree models exploit well.

3. New ATS features contributed modestly: home_ats_record_5g ranked
   6th in permutation importance. Rolling ATS records capture a
   real (but weak) signal about teams that are consistently beating
   or missing the spread.

4. Feature reduction helped: dropping 24 noisy features reduced
   overfitting and improved generalization by +0.21 percentage points.

5. The 2024-25 season (49.22% original, 48.33% v2) was particularly
   difficult -- the model's predictions were worse than coin-flip.
   This season accounted for most of the holdout underperformance.

6. Confidence filtering shows a slight edge at the 0.55 threshold
   (52.17%) but on only 414 games, which is not statistically
   significant.

7. Vegas lines are extremely efficient. The fundamental challenge
   is that spreads are set by sophisticated market-making algorithms
   and adjusted by sharp money, leaving very little exploitable
   edge for statistical models.

HONEST ASSESSMENT
-----------------------------------------------------------------
The 51.41% holdout accuracy is below the 52.38% breakeven. This
model would lose money at standard -110 vig if used for blind
flat betting. The in-sample metrics (52.57% overall) are inflated
by model training on that data.

The most promising angle is value-bet filtering (53.05% hit rate
at 5% edge threshold), but this relies on the model's probability
estimates being calibrated -- which is questionable given the
weak holdout performance.

No model in this experiment achieved 52.4%+ holdout accuracy.
The model is saved as an incremental improvement (+0.21pp) over
v1, but it should NOT be relied upon for profitable ATS betting.
